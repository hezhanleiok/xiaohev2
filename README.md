# xiaohev2

一个简洁、美观、可安装的**桌面和移动客户端**。本项目让用户能够管理自己的服务器配置，所有数据均安全地存储在用户本地。

## ✨ 功能特性

-   **跨平台支持**: 可在 **Windows, macOS, Linux, Android, 和 iOS** 上运行。
-   **完全独立与隐私**: 无需在电脑上运行任何辅助软件。所有服务器数据都只存在于您自己的设备上。
-   **多种导入方式**: 支持手动输入、从剪贴板导入、从订阅URL导入以及通过摄像头扫描二维码添加节点或订阅。
-   **智能国家识别**: 添加服务器时，应用会自动根据服务器IP识别其所在国家，并显示对应的国旗图标。
-   **本地持久化**: 您的所有配置都会安全地保存在设备本地，关闭或刷新页面后数据依然存在。
-   **自动分组**: 应用会根据您为服务器设置的“分组”名称，自动将服务器列表进行归类和展示。
-   **主题切换**: 内置设置中心，支持浅色（Light）和深色（Dark）主题一键切换。
-   **PWA 支持**: 也可以作为 PWA (Progressive Web App) 添加到手机主屏幕或桌面。

## ✅ 支持的平台

您可以从 [GitHub Releases](https://github.com/hezhanleiok/xiaohev2/releases) 页面下载适用于您操作系统的安装包。

-   **桌面端**:
    -   Windows (`.exe`)
    -   macOS (`.dmg`)
    -   Linux (`.AppImage`, `.deb`)
-   **移动端**:
    -   Android (`.apk`)
    -   iOS (需要自行签名安装)

## 🚀 如何使用

下载并安装应用后，您可以开始添加您的服务器配置。

### 添加服务器

点击应用主界面右上角的 **“+”** 号按钮，会弹出一个添加服务器的窗口。您有四种方式来添加：

1.  **从剪贴板导入**:
    -   提前复制好您的服务器节点链接（如 `vmess://...`）或订阅链接。
    -   选择 **“剪贴板”** 标签页，点击“从剪贴板导入”按钮，应用会自动识别并添加。

2.  **从 URL 导入**:
    -   选择 **“URL”** 标签页。
    -   将您的订阅链接粘贴到输入框中。
    -   点击“从URL导入”按钮。

3.  **扫码导入**:
    -   选择 **“扫码”** 标签页。
    -   授予应用相机权限。
    -   将服务器节点或订阅的二维码对准扫描框，即可自动识别并添加。

4.  **手动输入**:
    -   选择 **“手动”** 标签页。
    -   依次填写服务器的备注、地址、端口、协议等信息。
    -   点击“保存”按钮。

### 主界面操作

-   **选择节点**: 单击任意一个服务器节点来选中它。
-   **连接/断开**: 点击屏幕下方巨大的圆形 **“连接”** 按钮来启动或停止服务。
-   **测试延迟**: 点击底部的 **“测试延迟”** 按钮可以测试所有节点的网络延迟。您也可以点击分组标题旁的闪电图标，只测试该分组下的节点。
-   **排序**: 点击底部的 **“默认排序”** 按钮，可以在“默认”、“按延迟”、“按名称”三种排序方式之间切换。
-   **设置**: 点击主界面右上角的 **齿轮** 图标，可以进入设置页面，进行主题切换等操作。
