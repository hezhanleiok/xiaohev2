# xiaohev2

一个简洁、美观的客户端，用于管理您的服务器配置。所有数据都安全地存储在您自己的设备上。

## ✨ 主要功能

-   **全平台支持**: 可在 **Windows, macOS, Linux, Android, 和 iOS** 上运行。
-   **注重隐私**: 所有服务器数据只存在于您自己的设备上，不会上传到任何地方。
-   **多种导入方式**: 支持手动输入、从剪贴板、订阅链接以及扫描二维码添加。
-   **智能识别**: 自动识别服务器所在国家，并显示国旗图标。
-   **自动分组**: 根据您设置的“分组”名称，自动对服务器列表进行分类。
-   **主题切换**: 内置浅色（Light）和深色（Dark）主题，一键切换。

## ✅ 支持的平台

您可以从项目的发布页面下载适用于您操作系统的安装包。

-   **桌面端**:
    -   Windows (`.exe`)
    -   macOS (`.dmg`)
    -   Linux (`.AppImage`)
-   **移动端**:
    -   Android (`.apk`)
    -   iOS

## 🚀 如何使用

下载并安装应用后，您可以开始添加您的服务器配置。

### 添加服务器

点击主界面右上角的 **“+”** 号，有四种方式添加：

1.  **从剪贴板导入**: 提前复制好节点或订阅链接，点击“从剪贴板导入”即可。
2.  **从 URL 导入**: 将您的订阅链接粘贴到输入框中，然后点击导入。
3.  **扫码导入**: 将服务器节点或订阅的二维码对准扫描框即可。
4.  **手动输入**: 依次填写服务器的各项信息并保存。

### 主界面操作

-   **选择节点**: 单击任意一个服务器来选中它。
-   **连接/断开**: 点击屏幕下方巨大的圆形 **“连接”** 按钮来启动或停止服务。
-   **测试延迟**: 点击底部的 **“测试延迟”** 按钮可测试所有节点的网络延迟。您也可以点击分组标题旁的闪电图标，只测试该分组下的节点。
-   **排序**: 点击底部的排序按钮，可以在“默认”、“按延迟”、“按名称”三种排序方式之间切换。
-   **设置**: 点击右上角的 **齿轮** 图标，可以进行主题切换等操作。
