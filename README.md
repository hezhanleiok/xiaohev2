# xiaohev2

一个简洁、美观、可安装的**桌面和移动客户端**。本项目让用户能够管理自己的服务器配置，所有数据均安全地存储在用户本地。

利用 CapacitorJS，这个项目将一个现代 Web 应用打包成一个功能完整的、跨平台的原生应用。

## ✨ 功能特性

-   **跨平台支持**: 一份代码，可打包成 **Windows, macOS, Linux, Android, 和 iOS** 应用。
-   **完全独立与隐私**: 无需在电脑上运行任何辅助软件。所有服务器数据都只存在于您自己的设备上。
-   **多种导入方式**: 支持手动输入、从剪贴板导入、从订阅URL导入以及通过摄像头扫描二维码添加节点或订阅。
-   **智能国家识别**: 添加服务器时，应用会自动根据服务器IP识别其所在国家，并显示对应的国旗图标。
-   **本地持久化**: 您的所有配置都会安全地保存在设备本地，关闭或刷新页面后数据依然存在。
-   **自动分组**: 应用会根据您为服务器设置的“分组”名称，自动将服务器列表进行归类和展示。
-   **主题切换**: 内置设置中心，支持浅色（Light）和深色（Dark）主题一键切换。
-   **PWA 支持**: 依然支持作为 PWA (Progressive Web App) 添加到主屏幕。

## ⚙️ 技术架构

本应用使用 **React** 构建用户界面，并使用 **CapacitorJS** 将其打包成原生应用。

**React Web App (UI) ==> CapacitorJS (Native Bridge) ==> Desktop/Mobile App**

> **注意**: 当前版本仅完成了服务器的本地管理和 UI 交互。下一步是通过 Capacitor 的原生插件能力，集成一个底层网络核心来实现真正的“一键连接”功能。
